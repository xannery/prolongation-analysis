# Анализ эффективности пролонгации договоров — Topface Media

## 🎯 Цель проекта

Руководитель отдела сопровождения клиентов хотел получить объективную и детальную картину, насколько эффективно аккаунт-менеджеры справляются с одной из ключевых задач — пролонгацией договоров с клиентами.

**Задача:** рассчитать два официальных коэффициента пролонгации (в первый месяц и через месяц) за весь 2023 год, построить понятный управленческий отчёт и выявить ключевые зоны роста.

---

## 📊 Исходные данные

- **prolongations.csv** — список завершённых проектов с указанием последнего месяца реализации и ответственного аккаунт-менеджера (первичный источник AM).
- **financial_data.csv** — фактические суммы отгрузок по месяцам (с дублями частей оплаты, значениями «стоп», «end», «в ноль»).

---

## 🔧 Что было сделано

### 1. Глубокая предобработка данных
- Приведение названий месяцев к единому формату
- Корректная обработка специальных значений (`стоп`, `end`, `в ноль`)
- Агрегация дублей по проектам (одна строка = один проект)
- Правильное исключение проектов по правилам ТЗ

### 2. Расчёт коэффициентов пролонгации
- Два коэффициента по каждому месяцу 2023 года
- Расчёт на уровне «Весь отдел» и по каждому менеджеру
- Полностью соответствует бизнес-логике, описанной в ТЗ

### 3. Аналитический отчёт в Excel
- **Лист «Весь отдел»** — строго по шаблону ТЗ
- **Лист «Менеджеры за год»** — годовые итоги по каждому менеджеру
- **Лист «Менеджеры по месяцам»** — динамика коэффициентов помесячно
- Дополнительные листы с визуализациями

### 4. Визуализация
- Два **heatmap** (1-й и 2-й коэффициент) с сортировкой менеджеров по эффективности

## 📈 Ключевые insights и выводы

**Общие результаты по отделу за 2023 год:**
- Средний коэффициент пролонгации **в первый месяц** составил **0.5586** (55.86%).
- Средний коэффициент пролонгации **через месяц** значительно ниже — **0.0663** (всего 6.63%). Это главная зона роста для всего отдела.
- Лучший месяц по пролонгации в первый месяц — **Октябрь 2023** (коэффициент **0.8277**).
- Худший месяц — **Июнь 2023** (коэффициент **0.2210**).

**По менеджерам (годовые итоги):**
- **Лидер по объёму и эффективности** — **Васильев Артем Александрович** (121 проект, коэффициент 1-го месяца **0.5143**).
- Самый высокий средний коэффициент 1-го месяца показал **Смирнова Ольга Владимировна** (**0.7004**).
- **Соколова Анастасия Викторовна** также демонстрирует сильный результат (**0.6089**).
- У **Ивановой Марии Сергеевны** и **Поповой Екатерины Николаевны** коэффициент 2-го месяца близок к нулю, что говорит о практически полном отсутствии пролонгации «через месяц».

**Важные наблюдения:**
- Коэффициент пролонгации через месяц остаётся крайне низким у большинства менеджеров (многие месяцы = 0.0000). Это системная проблема, а не индивидуальная.
- Наблюдается высокая волатильность коэффициентов от месяца к месяцу — особенно заметно у Васильева и Соколовой.
- В некоторых месяцах (например, октябрь и март) отдел показывает очень хорошие результаты, что говорит о наличии потенциала и успешных практик, которые можно масштабировать.

**Рекомендации для руководителя отдела:**
1. Сфокусировать внимание на улучшении пролонгации «через месяц» — даже небольшое повышение этого коэффициента даст значительный прирост выручки.
2. Провести разбор лучших практик Васильева и Смирновой и внедрить их в работу остальных менеджеров.
3. Изучить причины провалов в июне и сентябре 2023 года.
4. Рассмотреть введение индивидуальных KPI по обоим коэффициентам пролонгации.
